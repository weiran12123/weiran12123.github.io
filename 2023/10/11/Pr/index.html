Pr快捷键

  • 素材导入：Ctrl+i
  • 媒体导出：Ctrl+M
  • 新建项目：Ctrl + Alt + N
  • 新建序列:Ctrl + N
  • 保存:Ctrl + S
  • 另存为：Ctrl + Alt + S
  • 播放暂停：空格
  • 快退 停止 快进： J K L
  • 慢速前后穿梭：Shift + L、Shift + J
  • 查找 反向查找：F、Shft + R
  • 入点 出点：i o
  • 跳到出入点：Shift + i 、 Shift + o
  • 取消出入点: Ctrl + Shift + i 、 Ctrl + Shift + q 都取消 Ctrl + Shift + X
  • 插入、覆盖：逗号、句号
  • 提升、提取：分号、引号 提升：删除出入点内容留下空的黑场 提取：删除出入点内容 ，后面的内容衔接上去
  • 导出帧：Ctrl + Shift + E
  • 移动、剃刀：V、C
  • 掐头、去尾：Q、W
  • 链接开关:Ctrl+L
  • 磁吸开关：S
  • 选中当前时间线所在片段 : D
  • 切断单轨：Ctrl + K
  • 切断多轨：Ctrl + Shift + K
  • 启用 禁用 ： Shift + E
  • 音量线加减：[ ]、左加右减
  • 打标记：M 标记记录:双击M
  • 跳到前面的标记：Shift + M
  • 跳到下一个标记：Ctrl + Shift + M
  • 取消当前标记：Ctrl + Alt + M
  • 撤销、还原：Ctrl + Z、Ctrl + Shift + Z
  • 复制：Alt 鼠标长按 传统办法：Ctrl + V、Ctrl + C
  • 复制粘贴属性：Ctrl + C、Ctrl + Alt + V (属性)
  • 剪辑点跳跃：上下
  • 帧数跳跃：左右 左右+Shift 5帧5帧增加
  • 添加默认视频转场：Ctrl + D
  • 添加默认音频转场：Ctrl + Shift + D
  • 添加默认视频音频转场：Shift + D
  • 窗口全屏：~
  • 缩放时间轴：加减号
  • 快速缩放：反斜杠
  • 鼠标缩放时间轴：Alt 滑轮
  • 前后穿梭时间轴：Shift 滑轮
  • 多机位切换摄像机：1 2 3 4
  • 互换片段位置：Ctrl + Shift + 鼠标左键拖拽
  • 上下移动片段所在轨道：选中素材,按住Alt上下箭头变换素材所在轨道 不建议用Z
  • 按住Shift拖拽素材快速对齐到某个位置：如标记点、出入点、剪辑点
  • 工具栏快捷键
  • 向前选择:A
  • 向后选择:Shift + A
  • 波纹编辑：B
  • 滚动编辑：N
  • 比例拉伸工具：R
  • 内滑、外滑：U、Y
  • 文字工具:T
